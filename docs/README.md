# Projekt-Dokumentation

Diese Dokumentation hält den aktuellen Stand, Entscheidungen und ToDos zum Projekt fest.

## Schnellzugriff

- **Aktueller Stand:** `status/current.md`
- **ToDos / Backlog:** `todos/backlog.md`
- **Entscheidungen (ADR):** `decisions/`
- **Chat-Protokolle:** `chatlogs/`
- **Vorlagen:** `templates/`

## Struktur

- `docs/status/` – aktueller Projektstand, technische Hinweise, offene Punkte
- `docs/todos/` – Backlog und Aufgabenliste
- `docs/decisions/` – Architektur- und Produktentscheidungen (ADR)
- `docs/chatlogs/` – relevante Chat-Zusammenfassungen/Protokolle
- `docs/templates/` – Vorlagen für neue Einträge
